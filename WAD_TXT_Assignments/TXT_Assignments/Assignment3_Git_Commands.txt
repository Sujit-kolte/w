STEPS TO CREATE GITHUB REPO:
1. Go to https://github.com
2. Click New Repository
3. Name it wad-assignments
4. Keep it Public
5. Do NOT initialize with README
6. Copy the repo URL
7. Use it in: git remote add origin <URL>
8. Then run: git push -u origin main
